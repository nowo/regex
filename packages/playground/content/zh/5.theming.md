# 主题与配色

每个渲染出的 SVG 都用 `--rr-*` 这组 CSS 自定义属性来着色，并把亮色默认值内联在
SVG 里，因此开箱即用、无需额外 CSS。想换色或加暗色主题，只要在包裹 SVG 的任意
元素上设置这些变量即可——你只需覆盖想改的那几个，无需重新渲染。

## 语法配色

每个语法类别对应一种颜色，源码条、语法参考和铁路图的强调色共用它，所以同一个
token 在各处颜色一致。下面是 `--rr-syntax-*` 变量及其亮色默认值。

::palette-table
::

```css
/* 亮色默认值已内联在 SVG 里；只需在包裹 SVG 的元素上覆盖你想改的变量。 */
.regex-diagram {
  --rr-syntax-literal: #3b82f6;     /* a b 1            */
  --rr-syntax-charset: #16a34a;     /* \d \w .          */
  --rr-syntax-class: #d97706;       /* [a-z] [^0-9]     */
  --rr-syntax-anchor: #8b5cf6;      /* ^ $ \b           */
  --rr-syntax-quantifier: #c026d3;  /* * + ? {n,m}      */
  --rr-syntax-group: #e11d48;       /* ( ) (?<name>     */
  --rr-syntax-lookaround: #ea580c;  /* (?= (?! (?<=     */
  --rr-syntax-backref: #0d9488;     /* \1 \k<name>      */
  --rr-syntax-alternation: #64748b; /* a|b|c            */
}
```

## 结构与暗色

除配色外，这些变量控制轨道、文字和标签——暗色主题主要就调它们——以及源码条的
悬停高亮。

```css
.dark .regex-diagram {
  --rr-rail: #64748b;   /* 连接轨道          */
  --rr-text: #e2e8f0;   /* 源码条与盒内文字   */
  --rr-muted: #94a3b8;  /* 分组 / 类标签      */
  --rr-group: #64748b;  /* 中性分组描边       */
}

/* 源码条悬停高亮：底色浓度与光晕扩散。 */
.regex-diagram {
  --rr-syntax-hl-opacity: 0.18;
  --rr-syntax-glow: 8px;
}
```
